# Julie - the chatbot that knows you

## How to install and run

### For Windows
```
git clone https://github.com/VIVelev/OldAndBald
cd ./OldAndBald
python -m pip install -r requirements.txt
cd ./elasticsearch
bin/elasticsearch
cd .. && python home.py
```

### For Max OSX and Linux
```
git clone https://github.com/VIVelev/OldAndBald
cd ./OldAndBald
pip3 install -r requirements.txt
cd ./elasticsearch
bin/elasticsearch
cd .. && python3 home.py
```
